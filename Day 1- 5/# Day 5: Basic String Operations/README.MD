# Day 5: Basic String Operations

## ✅ What I Did
- Performed basic string manipulations
- Explored methods like `upper()`, `lower()`, `replace()`, `strip()`
- Used string slicing and length

## 🧠 Key Concepts
- `"hello".upper()` → `"HELLO"`
- `"  hi  ".strip()` → `"hi"`
- `"Python"[0:3]` → `"Pyt"`
- `len("hello")` → `5`
- `"Py" in "Python"` → `True`
